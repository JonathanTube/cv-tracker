# cv-tracker

Help you manage and track your CV applications.

# tech stack

- Vanilla Javascript
- Node
- React
- TailwindCSS
- Vite
- Express

# Installation

1. Clone the repository

```
git clone https://github.com/yourusername/cv-tracker.git
```

2. Install dependencies

```
pnpm install
```

3. Run the application as development mode

```
pnpm dev
```
